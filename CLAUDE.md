# דשבורד שלי — Personal Health OS

דשבורד בריאות אישי לעומר: אימונים, תזונה טבעונית, הרגלים והתקדמות שבועית. בעברית, RTL, mobile-first.

## סטאק
- **קובץ יחיד**: `index.html` — HTML + CSS + Vanilla JS, בלי build ובלי תלויות npm.
- **נתונים**: `localStorage` בלבד (מפתח `fitos-omer-v1`). אין שרת.
- **פונט**: Heebo (Google Fonts CDN). **אייקונים**: Phosphor Web (CDN, regular + fill).
- **תצוגה מקדימה**: `.claude/launch.json` → שרת `dashboard` (npx http-server, פורט 4173).

## שפת עיצוב (לא לשבור)
- דארק נעול: רקע `#0B0D0A`, כרטיסים `#13160F`, accent יחיד ליים `#B9E94E`. בלי סגול, בלי גרדיאנטים, בלי glow.
- רדיוסים: כרטיסים 20px, כפתורים pill, קלטים 14px.
- בלי אימוג'ים ובלי קווים מפרידים ארוכים (em-dash) בטקסטים.
- Dial-ים (taste-skill): VARIANCE 5 / MOTION 4 / DENSITY 4.

## לוגיקה חשובה
- **תוכנית אימונים**: קליסטניקס (משקל גוף) של אליה בונן, שלב הבסיס רמה 3.5. **4 כוח + 2 אירובי + מנוחה**. ראשון=דחיפה(A), שלישי=משיכה(B), רביעי=רגליים, שישי=אימון הכוח הרביעי שמתחלף A/B לפי `weekIndex % 2`. שני+חמישי=אירובי, שבת=מנוחה. תרגיל = `{n, d, group?}` כש-`group` מקבץ סופרסטים סמוכים. התוכנית ב-`state.program` עם `version` (PROG_VERSION); שינוי מבני → להעלות גרסה ולהחליף ב-`load()`. אסור להמציא תרגילים מחוץ לרפרנס של בונן.
- **מחולל תפריט**: צהריים ~33% ומערב ~30% מיעד הקלוריות, עם העדפת חלבון. מאגר `MEALS` טבעוני בלבד.
- **יעדים (ברירת מחדל)**: 2100 קק"ל, 120 ג' חלבון, 10 כוסות מים, 10,000 צעדים, שינה 7.5. משקל: 68 → 65 ק"ג. הכול ניתן לעריכה בהגדרות.
- שבוע מתחיל ביום ראשון. רצפים לא נשברים על "היום" שעוד לא הושלם.
